## What is Bartender?

**Bartender** lets you hide, show, reorder, and group the icons in your Mac's menu bar. When many apps add icons (Wi‑Fi, battery, cloud, VPN, music, etc.), the bar gets crowded. Bartender stores extra icons in its own bar or dropdown and shows them on click or via keyboard shortcut. Supports **macOS Big Sur through Sonoma**, **Intel and Apple Silicon**, including MacBooks with a notch.

---

## Notch and Menu Bar Support

Bartender is built to work with the split menu bar: you can hide less important icons so that Wi‑Fi, battery, and your most-used items stay visible on either side of the notch, and the rest appear in the Bartender bar when you click or use the shortcut. You choose which icons stay in the main bar and which move to Bartender. No system icons are removed—they are only hidden or shown in Bartender’s bar. After major macOS updates, install the latest Bartender so it stays compatible with any menu bar changes.

---

## Menu Bar Visibility Options

Bartender gives each icon one of a few visibility modes:

- **Always show in Menu Bar** — The icon is always visible in the main menu bar (e.g. Wi‑Fi, battery, clock).
- **Show in Bartender Bar** — The icon is hidden from the main bar and appears only when you open the Bartender bar (click the Bartender icon or use the shortcut). Good for rarely used items.
- **Show when the Menu Bar is clicked** — The icon appears in the main bar when you click anywhere in the menu bar (macOS behavior). Saves space but keeps the icon one click away.
- **Always hide** — The icon never appears in the main bar; it's only in the Bartender bar. Use for items you want to keep but almost never need at a glance.

You can **reorder** icons by dragging in Bartender's settings so that when they're visible, they appear in your preferred order (e.g. Wi‑Fi and battery first, then Bartender's overflow).

## Notch & Multi-Display Behavior

**MacBook with notch:** On 14" and 16" MacBook Pro (and similar), the menu bar has less space next to the notch. Bartender is especially useful there: hide less important icons so the ones you need (e.g. Wi‑Fi, battery, Spotlight) stay visible without crowding. The Bartender bar can be set to appear on the left or right of the notch area depending on version and settings.

**Multiple displays:** You can choose which display shows the main menu bar (usually the one with the menu bar) and where the Bartender bar appears. Check **Bartender → Settings** (or **Preferences**) for options like "Show Bartender bar on" and per-Space behavior so it matches your setup.

## Tips & Best Practices

- **Hide** rarely used icons first; put them in the Bartender bar and set a shortcut you'll remember (e.g. **Option + B**) to open it.
- Use **Reorder** to keep Wi‑Fi, battery, and favorites in fixed positions.
- On **multi-display** setups, review which display shows the Bartender bar.
- After **macOS updates**, install Bartender updates for full compatibility with menu bar changes.
